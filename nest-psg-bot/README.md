# Telegram bots Nest.js

Небольшой бот разработанный на фреймворке Nest.js
Практика в создании бота, использования мощного фреймворка и TypeScript'а в работе

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
